제외사항 : 
  1. JOIN 구문 앞에 LEFT, RIGHT 등 안 붙이면 정렬 제외
  2. 서브쿼리 안의 UNION 정렬 제외
  3. CTE 구문, 되도록 주석은 ()안에 사용 권장
  4. SELECT 첫 컬럼이 CASE 문은 정렬 제외
